# ☀️ Tutor Solar ZNI - Asistente Inteligente Offline
Este proyecto es un chatbot basado en IA diseñado para funcionar en **Zonas No Interconectadas (ZNI)** de Colombia, donde el acceso a internet es limitado o inexistente.

## 🛠️ Características
- **100% Offline:** Utiliza Ollama (Phi-3) y AnythingLLM localmente.
- **Base de Conocimientos:** Entrenado con normativas de la CREG, manuales del IPSE y guías de mantenimiento solar.
- **Interfaz Amigable:** Desarrollado con Streamlit para una navegación intuitiva.

## 🚀 Cómo ejecutarlo
1. Instala AnythingLLM y Ollama.
2. Clona este repo: `git clone https://github.com/tu-usuario/tutor-solar-zni`
3. Instala requerimientos: `pip install -r requirements.txt`
4. Ejecuta: `streamlit run tutor_solar.py`
